System Components

A breakdown of the physical IoT infrastructure powering the data collection and
transmission.

C++ Microcontroller

ESP32 / NodeMCU acts as the brain, processing analog inputs and pushing JSON payloads to the web via Wi-Fi.

Telemetry Sensors

Dual INA219 modules for precision-measuring the DC voltage and current from both generation sources.

DC Generators

Photovoltaic cells and a micro wind turbine motor equipped with a rectifier circuit to stabilize output.
